checkfileseq -- scan directories for file sequences with missing files

checkfileseq is a command line tool for scanning directories that contain file sequences (e.g. file.001.png, image.002.png, ... image.NNN.png, etc.) and determine if each file sequence present is complete.

It defines one class FileSequenceChecker which does all the processing.
